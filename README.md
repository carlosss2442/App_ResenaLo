## Descripción del proyecto

**ReseñaLO** es una aplicación híbrida de red social y Google Maps que permite a los usuarios descubrir, evaluar y compartir sus experiencias en lugares de interés que normalmente no tendrían la misma visibilidad. La idea es facilitar la búsqueda de lugares interesantes a través de reseñas y opiniones de otros usuarios, mejorando la toma de decisiones a la hora de decidir dónde ir.

La aplicación permite a los usuarios añadir lugares, dejar reseñas, calificarlos y ver opiniones de otras personas, creando así una comunidad activa que puede interactuar y recomendar nuevos lugares para explorar. Este enfoque hace que **ReseñaLO** sea más que una simple aplicación de mapas, ya que permite que cada experiencia sea enriquecida con la retroalimentación de la comunidad.

## Objetivo del proyecto

El objetivo de **ReseñaLO** es crear una herramienta útil para jóvenes y adultos para descubrir lugares menos conocidos pero interesantes para visitar, como miradores, bares, museos, eventos locales, etc. También buscamos facilitar la toma de decisiones mediante reseñas personalizadas, mejorando así la experiencia de usuario en el descubrimiento de nuevos lugares.

## Project Description

**ResenaLO** is a hybrid application that combines a social network with Google Maps, allowing users to discover, evaluate, and share their experiences in places of interest that would not normally have the same visibility. The idea is to make it easier to find interesting places through reviews and opinions from other users, improving decision-making when choosing where to go.

The application allows users to add places, leave reviews, rate them, and view opinions from others, creating an active community that can interact and recommend new places to explore. This approach makes ReseñaLO more than just a map application, as it allows each experience to be enriched with community feedback.

## Project Objective

The objective of ReseñaLO is to create a useful tool for young people and adults to discover lesser-known but interesting places to visit, such as viewpoints, bars, museums, local events, etc. We also aim to facilitate decision-making through personalized reviews, thereby improving the user experience in discovering new places.
